# 高考日语真题词频库

这是一个从 1998-2025 年高考日语真题派生出的词频数据项目。它把词频拆成了总览、阅读/解题、听力题面、听力听到内容、听力综合几个视角，方便考生优先看高频词，也方便研究者下载 CSV 做进一步分析。

## 在线浏览

发布 GitHub Pages 时，建议选择本仓库的 `docs/` 目录作为发布源。发布后打开 Pages 地址即可浏览静态词库。

本地预览可以运行：

```bash
python -m http.server 8000 --directory docs
```

然后打开 `http://localhost:8000`。

## 项目结构

```text
docs/                 # GitHub Pages 静态词库
docs/data/            # 网页使用的轻量 JSON
data/                 # 可下载 CSV / 验证报告 / 脱敏 ASR manifest
```

## 数据范围

- 阅读/解题：来自试卷文本中的解题部分。
- 听力题面：来自试卷上印出的听力题文字。
- 听力听到内容：来自听力音频或听到内容的转写，其中 2002-2019 主要为机器 ASR。
- 听力综合：听力题面 + 听力听到内容。

## 质量说明

听力听到内容中的机器 ASR 数据来自自动语音识别，未完全人工校对；适合观察听力负荷和高频听感词，不应视为完全等同于官方文本。

已知提示：

- 2002-2005 纸面听力题面按用户范围决策跳过。
- 2004 听力音频 ASR 覆盖偏低，相关听力听到内容词频可能被低估。

页面中凡是包含机器 ASR 的词条都会显示 `ASR` 标记；涉及低覆盖 ASR 的词条会显示质量提示。

## 版权与公开数据声明

本项目不提供真题原文、PDF、Word、音频或完整转写文本。公开数据仅包含词形、原形、读音、词性、频次、来源类别、年份、试卷级聚合统计与数据质量说明。

## 外部词典
