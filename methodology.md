# Methodology / 统计方法

本文档说明 `gaokao-ja-wordfreq` 的统计对象、去重口径、ASR 边界、质量校验和公开数据范围。它的目的不是证明数据“绝对正确”，而是让读者知道这些词频从哪里来、怎么算、哪里需要谨慎使用。

## 1. 项目目标与统计对象

本项目统计 1998-2025 年高考日语真题中的词汇出现情况，目标是提供一个可检索、可下载、可复核的派生词频数据集。

本项目适合用于观察高频趋势、制定备考优先级和查漏补缺；不等同于官方词表、官方听力文本或绝对难度排序。

## 2. 数据源范围

本项目使用 PDF、Word、原卷、解析版、网页整理版等材料作为候选来源。候选来源只用于交叉核对、文本抽取和构建统计数据，不会按文件数量直接累加统计。

公开包不包含：

- 原始 PDF
- Word 原卷或解析版
- 听力音频
- 完整真题文本
- 完整 ASR 转写文本
- 本地文件路径
- API key、模型调用配置中的密钥或其他敏感信息

公开包只包含：

- 派生词频
- 词形、原形、读音、词性
- 来源类别
- 年份
- 试卷级聚合计数
- 质量标记
- 少量来源标注的学习例句片段

## 3. 试卷去重与版本处理

同一年份、同一套试卷可能存在 PDF、Word、原卷、解析版、网页整理版等多个候选版本。

最终统计以 `paper_id` 为去重单位。同一 `paper_id` 下的多个候选版本会先合并、校验或择优，然后只计入一次。因此，某一年份同一套试卷不会因为同时存在 PDF 和 Word 而被重复统计。

## 4. 文本抽取与分区

文本按来源和考试功能分区。内部字段与页面展示名称大致对应如下：

| 内部字段 | 页面展示 | 说明 |
|---|---|---|
| `solving` | 阅读/解题 | 阅读、语言知识、选项、题干等解题文本 |
| `listening_question` | 听力题面 | 试卷上印出的听力题文字 |
| `listening_transcript_text` | 听力听到内容 | 已有文本稿或听到内容文本 |
| `listening_asr_approved` | 听力听到内容 / ASR | 历史遗留内部字段名，当前不表示人工批准 |

页面中的“听力综合”表示听力题面加听力听到内容；“总览”表示阅读/解题、听力题面和听力听到内容的合计。

## 5. 听力音频与 ASR

当前线上发布版本为 `machine_complete_release`。这表示当前公开包已经包含可用的完整机器处理结果，但不表示所有听力听到内容都经过人工转写或人工逐句校对。

当前 2002-2019 听力听到内容包含机器 ASR 转写，公开 `asr_manifest_public.csv` 中的相关状态为 `machine_provisional_not_manual_approved`。这些数据适合观察听力负荷和高频听感词，但不应视为官方听力文本。

注意：内部字段名 `listening_asr_approved` 是历史遗留字段名。在当前公开版本中，它不表示“人工批准”或“人工校对通过”。

2004 年听力音频 ASR 覆盖偏低，相关听力听到内容词频可能被低估。因此，不建议把 2004 年听力听到内容与其他年份做严格横向比较。在总览、听力综合等包含 ASR 的统计范围中，2004 年相关词频也可能受到这一缺口影响。

## 6. 分词、词形归并与 word_id

日语文本使用 SudachiPy 进行形态素分析，统计时保留表面形、词典形、标准化形、读音和词性等字段。网页展示中常见字段含义如下：

- `display_form`：页面优先展示的词形，接近实际出现形式。
- `lemma_form`：词典形或归并后的原形，用于把活用形式归到同一词条。
- `reading`：读音，页面以平假名展示。
- `dict_query`：外部词典查询用词，优先选择更适合学习者查词的形式。
- `word_id`：稳定内部标识，由归一化原形、词类等信息生成，不直接等同于表面形。

例如，`行っ` 这类表面形在页面上会提示优先查 `行く`；`し` 这类动词活用会尽量指向 `する`。这样既保留真实出现形式，也方便学习者查词。

## 7. 分类体系

页面按考试来源和词性两个维度组织：

- 来源维度：总览、阅读/解题、听力题面、听力听到内容、听力综合。
- 词性维度：实词、名词、动词、形容词、副词、代名词、连体词、接续词、感动词、助词、助动词、接头辞、接尾辞等。

其中“实词”主要覆盖名词、动词、形容词、副词、代名词、连体词、接续词、感动词等；助词、助动词和符号类不计入实词。

## 8. 次数统计口径

页面和 CSV 中的主要计数字段含义如下：

- 当前分类次数：当前页面分类下的出现次数。
- 总次数：总览范围内的出现次数。
- 试卷数：该词在当前范围内出现过的 distinct `paper_id` 数量。
- 年份分布：按年份汇总的出现次数。
- 每张试卷次数：按 `paper_id` 汇总的出现次数。

听力综合 = 听力题面 + 听力听到内容。

总览 = 阅读/解题 + 听力题面 + 听力听到内容。

## 10. 质量校验

当前公开版本由机器校验脚本检查后发布。当前验证摘要：

- 验证状态：`passed`
- 校验门数量：84
- hard failure：0
- soft warning：1
- 当前 soft warning：2004 听力音频 ASR 覆盖偏低；公开站点已标记为 `low_asr_coverage_2004`。

hard failure 用于阻止发布；soft warning 用于提示已知数据质量风险。当前主要 soft warning 是 2004 听力音频 ASR 覆盖偏低。

## 11. 词典释义、Moji 跳转与例句

页面中的中文短释义用于帮助中文母语学习者快速理解词条，不作为权威词典解释。释义生成时参考 JMdict / JMdict-simplified 的英文义项，并经过自动生成流程得到中文短释义；助词、助动词等高频功能词另有少量人工补充短释义。

由于自动生成可能存在简化、遗漏或多义项选择偏差，疑难词、近义词和考试关键表达建议继续使用 MOJi 或其他专业词典核对。

Moji 跳转优先使用 `dict_query`，而不是直接使用表面形。例如活用形、假名表记和常见特殊词会尽量跳到更适合学习者查询的词典形。

页面中的例句是少量来源标注的学习语境片段，用于帮助理解词条在真题中的出现方式；本项目不提供完整真题文本或完整 ASR 转写文本。

## 12. 已知缺口与使用建议

请注意：

- 2002-2005 纸面听力题面按当前范围决策跳过。
- 2004 听力音频 ASR 覆盖偏低，相关听力听到内容词频可能被低估。
- 机器 ASR 可能出现漏识别、误识别或分词偏差。
- 自动分词和原形归并可能存在个别错误。
- 中文短释义可能有多义项选择偏差。
- 词频不等于官方词表。
- 词频高不代表一定会考。
- 词频低不代表不用掌握。
- 听力 ASR 词频不等同于官方听力文本。
- 出现次数不等同于词汇难度。
- 本项目不能替代教材、词典或教师讲解。

## 13. 可下载文件说明

公开下载目录包括：

- `data/all/`：总览词频。
- `data/solving/`：阅读/解题词频。
- `data/listening/question/`：听力题面词频。
- `data/listening/heard/`：听力听到内容词频。
- `data/listening/comprehensive/`：听力综合词频。
- `data/metadata/`：统计范围、验证摘要和脱敏 ASR manifest。

每个范围目录通常包含：

- `wordfreq.csv`：按词条聚合的词频表。
- `wordfreq_by_paper.csv`：按词条和试卷聚合的词频表。

元数据目录包含：

- `scope_summary.csv`：各统计范围的词条数、试卷数、次数和 ASR 标记。
- `validation_summary.json`：公开验证摘要。
- `asr_manifest_public.csv`：脱敏后的 ASR manifest，不包含本地路径或完整文本。
